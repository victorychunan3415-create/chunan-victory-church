# 竹南勝利堂網站設計方案

## 方案一:有機溫暖主義 (Organic Warmth)

**Design Movement**: 受北歐設計與日式侘寂美學啟發,強調自然材質感與人性溫度

**Core Principles**:
- 柔和的曲線與圓角,避免生硬的直角設計
- 大量留白營造呼吸感,讓內容自然流動
- 以「人」為核心的視覺敘事,強調真實的團契互動
- 溫暖的色調搭配柔和的陰影,創造親近感

**Color Philosophy**:
- 主色 #F7F5F2 (米白):如同晨光灑落的溫柔,象徵接納與安定
- 輔色 #6B7F6A (橄欖綠):生命成長的隱喻,帶來平靜與希望
- 強調色 #C7A45D (暖金):信仰的核心光芒,用於重要行動呼籲
- 文字 #333333 (深灰):易讀不壓迫,保持專業感

**Layout Paradigm**:
- 非對稱流動式佈局,避免傳統的居中網格
- 採用「視覺呼吸」系統:內容區塊間距至少 80-120px
- 首頁採用左右交錯的卡片式排列,創造動態閱讀節奏
- 使用 2/3 + 1/3 黃金比例分割,而非對稱的 1/2

**Signature Elements**:
- 手繪風格的分隔線與裝飾元素(如橄欖枝、十字架輪廓)
- 柔和的漸層遮罩覆蓋在圖片上,確保文字可讀性
- 圓角卡片(border-radius: 24px)搭配細膩的陰影(shadow-sm)

**Interaction Philosophy**:
- hover 時元素輕微上浮(translateY: -4px)並增強陰影
- 頁面滾動時內容區塊以淡入方式逐步顯現
- 按鈕點擊有微妙的縮放回饋(scale: 0.98)

**Animation**:
- 使用 ease-out 曲線,模擬自然物理運動
- 淡入動畫持續時間 600-800ms,營造從容感
- 滾動視差效果應用於大型圖片背景,增加深度

**Typography System**:
- 標題:Playfair Display (serif, 優雅的襯線字體,傳遞信仰的莊重感)
- 內文:Noto Sans TC (無襯線,清晰易讀,適合中文長文)
- 標題層級:H1(48px/bold) → H2(36px/semibold) → H3(24px/medium)
- 行高:1.8 for body text,確保舒適的閱讀體驗

## 方案二:現代簡約主義 (Modern Minimalism)

**Design Movement**: 受瑞士平面設計與包豪斯啟發,追求功能至上的純粹美學

**Core Principles**:
- 極致的留白與網格系統,一切服務於資訊傳達
- 強烈的層級對比,透過字重與尺寸建立清晰結構
- 去裝飾化,讓內容本身成為設計的主角
- 精準的對齊與間距,展現專業與秩序

**Color Philosophy**:
- 主色 #F7F5F2 (米白):純淨的畫布,讓內容呼吸
- 輔色 #6B7F6A (橄欖綠):克制的點綴,用於導航與重要標籤
- 強調色 #C7A45D (暖金):極少量使用,僅用於 CTA 按鈕
- 文字 #333333 (深灰):主要文字色,搭配 #666666 作為次要資訊

**Layout Paradigm**:
- 嚴格的 12 欄網格系統,所有元素對齊網格線
- 首頁採用全寬區塊堆疊,每個區塊專注單一訊息
- 使用大量垂直空間(section 間距 160px),創造呼吸感
- 內容最大寬度 1200px,居中對齊

**Signature Elements**:
- 極細的分隔線(1px, opacity: 0.1)
- 大尺寸的留白區域作為視覺休息點
- 單色圖標系統,統一視覺語言

**Interaction Philosophy**:
- 微妙的 hover 狀態:僅改變透明度(opacity: 0.7)
- 快速的過渡動畫(200ms),保持俐落感
- 無多餘裝飾,互動回饋直接明確

**Animation**:
- 使用 linear 或 ease 曲線,避免過度彈性
- 動畫持續時間控制在 200-300ms
- 僅在必要時使用動畫,不干擾資訊獲取

**Typography System**:
- 全站使用 Inter (幾何無襯線,現代且中性)
- 標題層級:H1(64px/700) → H2(48px/600) → H3(32px/500)
- 嚴格的字重系統:Regular(400) for body, Medium(500) for subheadings, Semibold(600) for headings, Bold(700) for emphasis
- 行高:1.5 for body text,緊湊但不擁擠

## 方案三:溫暖敘事主義 (Warm Storytelling)

**Design Movement**: 受雜誌編排與故事性網頁設計啟發,強調情感連結與視覺敘事

**Core Principles**:
- 大膽的圖文混排,讓照片說故事
- 不對稱但平衡的佈局,創造視覺張力
- 豐富的層次與深度,透過重疊、陰影、模糊營造空間感
- 溫暖的色調與柔和的光影,傳遞歸屬感

**Color Philosophy**:
- 主色 #F7F5F2 (米白):如同紙張的質感,承載故事
- 輔色 #6B7F6A (橄欖綠):生命的隱喻,用於標題與重點區塊
- 強調色 #C7A45D (暖金):如同燭光,引導視線與行動
- 漸層應用:從米白到淡綠的柔和過渡,增加層次

**Layout Paradigm**:
- 雜誌式的多欄佈局,結合大圖與文字區塊
- 首頁採用「故事卷軸」設計:每個 section 是一個獨立場景
- 使用 z-index 創造元素重疊,增加深度感
- 內容區塊採用卡片式設計,但尺寸與位置不規則排列

**Signature Elements**:
- 大尺寸的英雄圖片(viewport height: 70-80vh)搭配深色漸層遮罩
- 浮動的卡片元素,帶有明顯陰影(shadow-lg)
- 手寫風格的引言文字或經文,增加人性溫度
- 圓形裁切的人物照片,強調「人」的核心

**Interaction Philosophy**:
- hover 時卡片放大(scale: 1.05)並增強陰影,營造「拾起」的感覺
- 滾動時背景圖片以不同速度移動(視差效果)
- 按鈕有明顯的 3D 按壓效果(active: translateY(2px))

**Animation**:
- 使用 ease-in-out 曲線,模擬自然的加速與減速
- 進場動畫採用「淡入+上移」組合(opacity + translateY)
- 動畫持續時間 400-600ms,營造流暢但不急促的節奏
- 滾動觸發動畫,讓內容逐步展開如同翻閱雜誌

**Typography System**:
- 標題:Cormorant Garamond (優雅的襯線字體,帶有文學氣質)
- 內文:Noto Sans TC (清晰易讀,與標題形成對比)
- 引言/經文:Dancing Script 或手寫風格字體(增加溫度)
- 標題層級:H1(56px/700) → H2(40px/600) → H3(28px/500)
- 行高:1.75 for body text,寬鬆舒適
- 字重對比強烈:標題 700,內文 400,形成清晰層級

## 最終選擇:方案三 - 溫暖敘事主義 (Warm Storytelling)

選擇理由:
1. 最符合「以人為主」的需求,透過大膽的圖文混排突出團契互動
2. 雜誌式編排能有效承載豐富的內容(講道、活動、小組等)
3. 溫暖的色調與深度感能吸引年輕族群,同時保持專業形象
4. 故事性的設計語言能有效傳達教會的異象與使命
5. 不對稱佈局避免傳統教會網站的刻板印象,更具現代感
