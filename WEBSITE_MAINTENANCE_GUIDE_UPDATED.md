## 快速開始

### 訪問網站

**開發環境**（本地測試）：
- 打開瀏覽器
- 訪問 `http://localhost:3000`

**生產環境**（正式網站）：
- 訪問您的 Cloudflare Pages 域名
- 例如：`https://chunan-victory-church.pages.dev`

### 登錄後台

1. 在網站右上角點擊「立即聯繫」或「登入」按鈕
2. 使用您的 Manus 帳號登錄
3. 登錄成功後，訪問 `/admin/news` 進入新聞管理後台

## 後台管理系統

### 後台訪問地址

```
http://localhost:3000/admin/news（開發環境）
https://your-domain.pages.dev/admin/news（生產環境）
```

### 權限說明

- **管理員**：可以新增、編輯、刪除所有內容
- **普通用戶**：只能查看前台內容

**如何設置管理員**：
1. 聯絡技術團隊或訪問資料庫管理界面
2. 在 `users` 表中找到您的帳號
3. 將 `role` 欄位改為 `admin`
4. 保存並重新登錄

## 最新消息管理

### 新增最新消息

#### 步驟 1：進入新聞管理後台

1. 登錄網站
2. 訪問 `/admin/news`
3. 點擊「新增新聞」按鈕

#### 步驟 2：填寫新聞信息

| 欄位 | 說明 | 示例 |
|------|------|------|
| **標題** | 新聞的主標題 | 「主日崇拜恢復實體聚會」 |
| **摘要** | 新聞的簡短描述（會顯示在列表中） | 「從 2 月 1 日起，我們恢復實體主日崇拜...」 |
| **內容** | 新聞的完整內容 | 使用富文本編輯器編寫 |
| **狀態** | 新聞的發佈狀態 | 草稿/已發佈/已歸檔 |

#### 步驟 3：使用富文本編輯器編寫內容

編輯器工具欄包含以下功能：

| 按鈕 | 功能 | 快捷鍵 |
|------|------|--------|
| **H1, H2, H3** | 標題（大、中、小） | - |
| **B** | 粗體 | Ctrl+B |
| **I** | 斜體 | Ctrl+I |
| **S** | 刪除線 | - |
| **•** | 項目符號列表 | - |
| **1.** | 有序列表 | - |
| **"** | 引用區塊 | - |
| **<>** | 代碼塊 | - |
| **🔗** | 插入連結 | - |
| **🖼️** | 上傳圖片 | - |
| **↶** | 撤銷 | Ctrl+Z |
| **↷** | 重做 | Ctrl+Y |

#### 步驟 4：上傳圖片

1. 在編輯器中點擊「🖼️」（圖片）按鈕
2. 在彈出的對話框中：
   - **點擊上傳**：點擊上傳區域選擇圖片
   - **拖拽上傳**：直接拖拽圖片到上傳區域
3. 選擇圖片後，系統會自動上傳到 CDN
4. 上傳完成後，圖片會自動插入編輯器

**圖片要求**：
- 格式：JPG、PNG、GIF、WebP 等
- 最大大小：5MB
- 建議寬度：800px 以上

#### 步驟 5：設置發佈狀態

在「狀態」下拉菜單中選擇：

- **草稿**：保存但不發佈，只有管理員可見
- **已發佈**：發佈到前台，所有訪客可見
- **已歸檔**：隱藏但保留記錄

#### 步驟 6：保存和發佈

1. 填寫完所有信息後，點擊「保存」按鈕
2. 系統會驗證表單數據
3. 如果有錯誤，會顯示紅色提示信息
4. 保存成功後，會返回新聞列表

### 編輯現有新聞

1. 進入新聞管理後台
2. 在列表中找到要編輯的新聞
3. 點擊該新聞的「編輯」按鈕
4. 修改相應內容
5. 點擊「保存」按鈕

### 刪除新聞

1. 進入新聞管理後台
2. 在列表中找到要刪除的新聞
3. 點擊該新聞的「刪除」按鈕
4. 確認刪除（此操作不可撤銷）

### 新聞列表視圖

後台新聞列表顯示以下信息：

| 欄位 | 說明 |
|------|------|
| **標題** | 新聞的主標題 |
| **狀態** | 當前發佈狀態（草稿/已發佈/已歸檔） |
| **建立時間** | 新聞建立的日期和時間 |
| **操作** | 編輯/刪除按鈕 |

## 其他內容管理

### 代禱事項

代禱事項由訪客通過「最新消息」頁面的表單提交。

**查看代禱事項**：
1. 訪客提交後，資料會自動保存到資料庫
2. 管理員可通過資料庫管理界面查看
3. 定期審查並為代禱事項禱告

### 活動報名

訪客可通過「最新消息」頁面的報名表單報名活動。

**查看報名信息**：
1. 訪客提交後，報名信息會自動保存
2. 管理員可通過資料庫管理界面查看報名者信息
3. 可導出報名列表用於統計

### 聯絡表單

訪客可通過「聯絡我們」頁面提交聯絡信息。

**查看聯絡信息**：
1. 聯絡信息會自動保存到資料庫
2. 管理員可通過資料庫管理界面查看
3. 定期回覆訪客的查詢

## 常見操作

### 操作 1：新增主日講道信息

1. 進入新聞管理後台
2. 點擊「新增新聞」
3. 填寫以下信息：
   - **標題**：「2026 年 X 月 X 日 主日講道」
   - **摘要**：「講題：[講題]，講員：[講員名字]」
   - **內容**：
     - 使用 H2 標題：「講題」
     - 輸入講題
     - 使用 H2 標題：「講員」
     - 輸入講員名字
     - 使用 H2 標題：「經文」
     - 輸入經文參考
     - 使用 H2 標題：「講道摘要」
     - 輸入講道內容
4. 上傳講道相關圖片
5. 設置狀態為「已發佈」
6. 點擊「保存」

### 操作 2：發佈活動公告

1. 進入新聞管理後台
2. 點擊「新增新聞」
3. 填寫以下信息：
   - **標題**：「[活動名稱] - 報名開始」
   - **摘要**：「[活動名稱]現已開放報名，詳情請見下文」
   - **內容**：
     - 使用 H2 標題：「活動名稱」
     - 使用 H2 標題：「時間」
     - 使用 H2 標題：「地點」
     - 使用 H2 標題：「報名方式」
     - 使用項目符號列表列出報名步驟
4. 上傳活動宣傳圖片
5. 設置狀態為「已發佈」
6. 點擊「保存」

### 操作 3：更新教會信息

教會的基本信息（聯絡方式、聚會時間等）存儲在代碼中。要更新這些信息：

1. 打開 Visual Studio Code
2. 打開 `client/src/pages/Home.tsx` 文件
3. 找到相應的信息部分
4. 修改內容
5. 保存文件
6. 提交到 GitHub（自動部署）

### 操作 4：更新首頁圖片

首頁的背景圖片和其他圖片存儲在 `client/public/images/` 目錄中。

**更新步驟**：
1. 準備新圖片（推薦大小：1920x1080px）
2. 將圖片放入 `client/public/images/` 目錄
3. 在代碼中更新圖片路徑
4. 提交到 GitHub

## 故障排除

### 問題 1：無法登錄後台

**症狀**：點擊登錄按鈕後沒有反應或跳轉到登錄頁面

**解決方案**：
1. 清除瀏覽器 Cookie：按 F12 打開開發者工具 → Application → Cookies → 刪除相關 Cookie
2. 重新打開網站並嘗試登錄
3. 確保您的帳號已被設置為管理員

### 問題 2：新聞無法保存

**症狀**：點擊「保存」後沒有反應或顯示錯誤信息

**解決方案**：
1. 檢查是否填寫了所有必填欄位（標題、摘要、內容）
2. 檢查網絡連接是否正常
3. 打開瀏覽器開發者工具（F12）查看錯誤信息
4. 如果仍有問題，聯絡技術支持

### 問題 3：圖片上傳失敗

**症狀**：上傳圖片時顯示錯誤信息

**解決方案**：
1. 檢查圖片大小是否超過 5MB
2. 確認圖片格式是否為 JPG、PNG、GIF 等
3. 檢查網絡連接
4. 嘗試使用其他瀏覽器
5. 如果仍有問題，聯絡技術支持

### 問題 4：網站無法訪問

**症狀**：訪問網站時顯示「無法連接」或「404 錯誤」

**解決方案**：
1. 檢查網絡連接
2. 確認域名是否正確
3. 等待 5-10 分鐘（可能正在部署）
4. 清除瀏覽器緩存（Ctrl+Shift+Delete）
5. 嘗試使用無痕模式訪問
6. 如果仍有問題，聯絡技術支持

### 問題 5：部署失敗

**症狀**：提交代碼到 GitHub 後，網站沒有更新

**解決方案**：
1. 檢查 GitHub Actions 的部署日誌
2. 確保所有環境變數都已正確設置
3. 檢查代碼是否有語法錯誤
4. 查看 Cloudflare Pages 的部署日誌
5. 如果仍有問題，聯絡技術支持
