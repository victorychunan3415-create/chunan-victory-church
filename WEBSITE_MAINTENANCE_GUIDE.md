## 簡介

## 修改最新消息

### 📍 最新消息文件位置

最新消息的內容存儲在以下文件中：

```
chunan-victory-church/
└── client/
    └── src/
        └── pages/
            └── News.tsx
```

### 步驟 1：打開最新消息文件

1. 用文本編輯器打開 `News.tsx` 文件
2. 您會看到一大堆代碼，但不用擔心！我們只需要修改特定的部分

### 步驟 2：找到新聞卡片部分

在文件中搜索（按 Ctrl+F 或 Cmd+F）以下文字：

```
newsItems
```

您會看到類似這樣的結構：

```javascript
const newsItems = [
  {
    id: 1,
    title: "新聞標題",
    date: "2026-01-30",
    category: "活動",
    image: "/images/news-1.jpg",
    excerpt: "新聞摘要文字...",
    content: "完整的新聞內容..."
  },
  {
    id: 2,
    title: "另一個新聞標題",
    ...
  }
]
```

### 步驟 3：修改新聞內容

#### 修改新聞標題：
找到 `title: "新聞標題"` 這一行，將引號內的文字改為您想要的標題。

**例子：**
```javascript
// 修改前
title: "新聞標題"

// 修改後
title: "主日聚會恢復正常時間"
```

#### 修改發佈日期：
找到 `date: "2026-01-30"` 這一行，改為今天的日期。

**日期格式：** `YYYY-MM-DD`（年-月-日）

**例子：**
```javascript
// 修改前
date: "2026-01-30"

// 修改後
date: "2026-01-31"
```

#### 修改分類：
找到 `category: "活動"` 這一行，可以改為：
- `"活動"` - 教會活動
- `"通知"` - 重要通知
- `"見證"` - 信仰見證
- `"公告"` - 一般公告

#### 修改新聞摘要：
找到 `excerpt: "新聞摘要文字..."` 這一行，改為簡短的摘要（1-2 句話）。

**例子：**
```javascript
excerpt: "本週主日聚會將於上午 10:30 開始，歡迎所有弟兄姐妹參加。"
```

#### 修改完整新聞內容：
找到 `content: "完整的新聞內容..."` 這一行，改為完整的新聞文章。

**例子：**
```javascript
content: "親愛的弟兄姐妹，感謝大家的耐心等待。本週主日聚會將恢復正常時間，上午 10:30 開始。敬請準時參加，一同敬拜讚美主。"
```

#### 修改新聞圖片：
找到 `image: "/images/news-1.jpg"` 這一行。

**圖片路徑說明：**
- `/images/news-1.jpg` 表示圖片在 `client/public/images/` 文件夾中
- 如果您上傳了新圖片，需要改為新圖片的文件名

### 步驟 4：添加新的新聞

如果您想添加新的新聞，在 `newsItems` 陣列中添加新的對象。

**模板：**
```javascript
{
  id: 3,
  title: "新聞標題",
  date: "2026-02-01",
  category: "活動",
  image: "/images/news-3.jpg",
  excerpt: "新聞摘要",
  content: "完整新聞內容..."
}
```

**重要：** 
- `id` 必須是唯一的數字（不能重複）
- 每個新聞項之間用逗號 `,` 分隔
- 最後一個新聞項後面**不要**加逗號

### 步驟 5：保存文件

按 **Ctrl+S**（Windows）或 **Cmd+S**（Mac）保存文件。

## 上傳和更換圖片

### 📍 圖片文件位置

所有網站圖片存儲在：

```
chunan-victory-church/
└── client/
    └── public/
        └── images/
```

### 步驟 1：準備您的圖片

1. 準備一張 JPG 或 PNG 格式的圖片
2. 建議圖片尺寸：
   - 新聞圖片：寬度 800px 以上
   - 首頁圖片：寬度 1200px 以上
3. 文件大小：不超過 5MB

### 步驟 2：上傳圖片到項目

1. 打開文件管理器
2. 進入 `chunan-victory-church/client/public/images/` 文件夾
3. 將您的圖片複製到這個文件夾中
4. **記住圖片的文件名**（例如：`news-3.jpg`）

### 步驟 3：在代碼中引用圖片

在 `News.tsx` 文件中，找到要修改的新聞項，改變 `image` 字段：

```javascript
// 修改前
image: "/images/news-1.jpg"

// 修改後
image: "/images/news-3.jpg"
```

### 步驟 4：保存並發佈

保存文件後，按照「發佈更改」部分的步驟發佈網站。

## 發佈更改

修改完最新消息和圖片後，需要發佈更改才能在網站上看到。

### 方法 1：使用 Manus 管理界面（推薦）

1. 登入 Manus 帳號
2. 進入您的項目
3. 點擊右側的 **「Preview」**（預覽）查看修改效果
4. 如果滿意，點擊 **「Publish」**（發佈）按鈕
5. 等待幾秒鐘，網站會自動更新

### 方法 2：使用 Git 上傳到 GitHub（進階）

如果您已經設置了 GitHub，可以通過以下步驟更新：

1. 打開終端/命令提示符
2. 進入項目文件夾：
   ```bash
   cd /path/to/chunan-victory-church
   ```

3. 添加所有修改：
   ```bash
   git add .
   ```

4. 提交修改：
   ```bash
   git commit -m "更新最新消息"
   ```

5. 上傳到 GitHub：
   ```bash
   git push
   ```

6. 在 Manus 中點擊 **「Publish」** 發佈

## 常見問題

### Q1：修改後網站沒有更新怎麼辦？

**A：** 
1. 確保您已經保存了文件（Ctrl+S 或 Cmd+S）
2. 確保您已經點擊了 **「Publish」** 按鈕
3. 嘗試刷新瀏覽器（F5 或 Ctrl+R）
4. 等待 30 秒後再檢查（網站可能需要時間更新）

### Q2：圖片上傳後沒有顯示

**A：** 
1. 檢查圖片文件名是否正確（區分大小寫）
2. 確保圖片路徑正確：`/images/filename.jpg`
3. 確保圖片文件確實在 `client/public/images/` 文件夾中
4. 嘗試使用不同的圖片格式（JPG 或 PNG）

### Q3：我不小心刪除了重要的新聞，怎麼恢復？

**A：** 
1. 如果您使用了 Git，可以查看之前的版本：
   ```bash
   git log
   ```
2. 找到之前的提交版本
3. 或者，您可以從 GitHub 上的備份恢復

### Q4：如何添加多個新聞？

**A：** 按照「步驟 4：添加新的新聞」部分操作。每次添加一個新聞項，確保：
- `id` 是唯一的
- 每個項之間用逗號分隔
- 最後一個項後面不要加逗號

### Q5：圖片應該用什麼格式？

**A：** 
- **JPG**：適合照片，文件較小
- **PNG**：適合圖標和透明背景，文件較大
- **WebP**：最新格式，文件最小（但不是所有瀏覽器都支持）

建議使用 **JPG** 或 **PNG**。

### Q6：新聞內容可以包含特殊格式（粗體、斜體等）嗎？

**A：** 可以！您可以使用 HTML 標籤：

```javascript
content: "這是<b>粗體</b>文字，這是<i>斜體</i>文字。"
```

常用的 HTML 標籤：
- `<b>粗體</b>` - 粗體
- `<i>斜體</i>` - 斜體
- `<u>下劃線</u>` - 下劃線
- `<br>` - 換行
- `<p>段落</p>` - 段落

## 📝 快速參考

### 修改新聞的步驟總結

1. 打開 `News.tsx` 文件
2. 找到 `newsItems` 部分
